MultiTerm: The database of the future 
 
Vasiliki Kanistra 
ABSTRACT  
MultiTerm is an electronic dictionary which is literally built by its own user. This application provides to 
the owner of the Trados program and user of the application all the necessary tools in order for him to 
be able to create easily a database which will include all the useful terms that a translator, an interpreter 
or any other person who uses terminology as part of his professional and everyday life need as well as 
other elements such as definitions, the sources of the terms, the context and so on. The goal of such an 
application is a rather easy and quick collection and access of data so as the user of the application will 
not have to search every term separately in the dictionary. Using MultiTerm one can have all the terms 
that one needs in one and only database.

διεκπεραίωση των εργασιών ενός μεταφραστή ή διερμηνέα αλλά και οποιουδήποτε άλλου
