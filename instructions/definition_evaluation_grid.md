# Instruction: What is a Definition? (Evaluation Grid)

A definition is the representation of a concept by a descriptive statement which serves to differentiate it from related concepts (ISO 1087-1:2000).

## Evaluation Criteria (Symbolic & Human)

| Criteria | Description | Score (1-5) |
|----------|-------------|-------------|
| **Conciseness** | Avoids redundant words or "circular" definitions. | |
| **Precision** | Clearly differentiates the concept from others. | |
| **Genus + Differentia** | Structure: [Superordinate concept] + [Specific characteristics]. | |
| **Language** | Native-like syntax, no machine translation artifacts. | |
| **RAG Accuracy** | Factually consistent with the source corpus. | |

## Examples of Correct Definitions
- **Celiac Disease**: A chronic immune-mediated disorder triggered by gluten ingestion in genetically predisposed individuals, causing inflammation of the small intestine.

## Symbolic Parser Checks
- [ ] Contains "genus" (is a [category])
- [ ] Length < 50 words
